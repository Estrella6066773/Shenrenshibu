# 找猫

阶段1：员工在失主家附近走访调查，看看是否有人见过猫的去向。

失败：很不幸，员工们采访的这位邻居今天并没有见到遗失的猫咪。你的员工打算再去问问其他邻居。

成功：有邻居看到猫被几个小孩追赶着逃向了废墟区。在邻居带领下，员工去寻找那几个小孩，打算让他们带路找到猫。

---

阶段2：你的员工顺利找到了那几个小孩。此时他们正围在一个笼子前，用树枝逗弄里面的两只老鼠。虽然找到了这几个小鬼，但这仅仅只是开始：这几个小鬼未必敢深入废墟区。若猫儿真的闯了进去，那麻烦可就大了。啧，该死的小鬼们。

失败：“我们没有见过那只猫咪。”年长的孩子这么说着，但他身后那几个小鬼的表情早已将他出卖。若施以影响，或许能让他们乖乖听话。

成功：你的员工设法让孩子们相信，那只猫不只是失主的宠物，还和失主认识的大人物关系匪浅。若猫真的走丢了，就算梁婶不主动刁难，那位大人物也不会轻易罢休。

---

阶段3：几个孩子悻悻地带你来到他们最后跟丢猫的地方——废墟区边缘一处废弃的通风管道。这条管道多半连通着废墟区，已经位于危险地带边缘。那只猫咪一定是在慌乱中钻了进去，才逃脱了这几个小鬼的魔爪。幸好这几个小鬼没有胆量跟进去，不然你接到的委托就不是找猫这么简单了。

失败：通风口荒废已久，但并非完全无人问津。你的员工没能在杂乱的动物足迹中辨别出猫的脚印，或许我们应该请更善于侦察的员工帮忙。

成功：你的员工在管道底端的沉积物中辨别出了崭新的脚印，并且不费吹灰之力就找到了脚印的主人。猫儿虽然有些惊慌，但好在没有应激。在员工的安抚下，小队很快就成功将猫收容。
