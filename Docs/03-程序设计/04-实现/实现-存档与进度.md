# 实现：存档与进度（SaveProgress）

| 字段 | 值 |
|------|-----|
| 模块键 | SaveProgress |
| 主代码路径 | `Assets/01_Scripts/Runtime/Save/`、`Assignment/Controller/CampaignSaveBackendAdapter.cs` |
| 程序集 | `ShenRenShibu.Runtime`、`ShenRenShibu.Domain` |

## 1. 职责边界

- **负责**：战役存档统一读写入口、槽位文件名、`schema_version` 校验占位。
- **不负责**：各模块字段序列化细节（仍由各 `ExportSaveData` / `ImportSaveData` 承担）。

## 2. 读档顺序（REQ-SAV-002）

1. PlayerEconomy
2. Employee（待接入）
3. Assignment
4. MessageSystem（MailInbox 等）
5. Conflict / Stress
6. StoryProgress（stub）

## 3. 核心类型

| 类型 | 说明 |
|------|------|
| `CampaignSaveOrchestrator` | 编排入口；快速存档/读档 |
| `ICampaignSaveBackend` | 后端接口 |
| `CampaignSaveBackendAdapter` | 包装 `AssignmentSaveReplayService` |
| `AssignmentSaveBundle` | 现有 JSON 包（见 Assignment 模块） |
