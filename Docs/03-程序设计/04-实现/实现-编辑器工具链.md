## 1. 脚本清单

| 文件 | 职责摘要 |
|------|-----------|
| `ShenRenShiBuAssetMenu.cs` | 菜单「神人事部/创建/*」：标签库、标签、简历模板、分支树、频道注册表、角色库、剧情、邮件数据库等；默认创建路径统一到 `Assets/04_Data/` |
| `AssignmentSystemSetupMenu.cs` | 「神人事部/场景/搭建委托系统」：生成委托根、模板组、运行时节点 |
| `AssignmentDailyRuntimeSetupMenu.cs` | 「神人事部/场景/一键配置日常委托运行时」；子菜单：安装发布器、**安装 Play 验收辅助**、日常占位模板、同步注册表 |
| `AssignmentDailyRefreshPublisher` | 运行时日常发布（REQ-ASG-011）；须与 `AssignmentManager` 同场景挂载 |
| `IdleAreaSetupMenu.cs` | 待命区场景搭建 |
| `AssignmentUat036TestRunnerMenu.cs` | 「神人事部/验收/运行全部 UAT-036 测试」；分项 EditMode / PlayMode |
| `AssignmentTemplateSOEditor.cs` | 委托模板 Inspector 扩展 |
| `ChatStorySOEditor.cs` | 剧情 SO 编辑器 |
| `MailEventDatabaseSOEditor.cs` | 邮件库编辑器 |
| `MailEventCsvToJsonConverter.cs` | CSV→JSON 转换；结果输出至 Console（**无** `DisplayDialog`） |
