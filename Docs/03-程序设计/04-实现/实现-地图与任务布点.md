# 实现：地图与任务布点（WorldMap）

| 字段 | 值 |
|------|-----|
| 模块键 | WorldMap |
| 主代码路径 | `Assets/01_Scripts/Presentation/WorldMap/` |
| 数据资产 | `Assets/04_Data/地图系统/WorldLocationCatalogSO` |

## 1. 职责边界

- **负责**：预置地点目录、`location_id` 查询、随机落点选取（stub）。
- **不负责**：开放世界地形；设定百科地点正文（`04-设定/真相/`）。

## 2. 核心类型

| 类型 | 说明 |
|------|------|
| `IWorldLocationCatalog` | 地点查询接口（Runtime 抽象） |
| `WorldLocationCatalog` | SO 驱动实现（Presentation） |
| `WorldLocationCatalogSO` | 地点表 ScriptableObject |
| `WorldLocationEntry` | 单条：id、显示名、世界坐标 |

## 3. 与委托集成（规划）

- 委托创建工厂读取 `location_id` 或随机选取（REQ-MAP-002）。
- 进行中委托标记显隐仍由 `AssignmentTaskBlockInteraction` 承担。
