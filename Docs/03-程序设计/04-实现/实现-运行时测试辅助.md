# 实现：运行时测试辅助（RuntimeTests）

| 字段 | 值 |
|------|-----|
| 模块键 | RuntimeTests |
| 关联设计 | `需求-运行时测试辅助.md` |
| 主代码路径 | `Assets/01_Scripts/Tests/Runtime/`、`Tests/PlayMode/`、`Tests/Editor/` |

## 1. 类型说明

| 类型 | 文件 | 说明 |
|------|------|------|
| `AssignmentTestBootstrap` | `AssignmentTestBootstrap.cs` | `DefaultExecutionOrder(-1400)`；可选门控 + `CreateAssignmentCommandEvent` |
| `AssignmentPlaymodeVerifier` | `AssignmentPlaymodeVerifier.cs` | 订阅委托相关事件并 `Debug.Log` |
| `AssignmentDailyRefreshPlaymodeVerifier` | `AssignmentDailyRefreshPlaymodeVerifier.cs` | 日结跨天 + 日常委托 `Created` 观察（UAT-036） |
| `AssignmentDailyRefreshUat036PlayModeTests` | `Tests/PlayMode/...PlayModeTests.cs` | UAT-036b 日结跨天 + 直接发布（2 用例） |
| `AssignmentDailyRefreshUat036EditModeTests` | `Tests/Editor/...EditModeTests.cs` | UAT-036a′ 无需 Play |
| `StartupInitAssignmentTestSender` | `StartupInitAssignmentTestSender.cs` | `DefaultExecutionOrder(-1000)`；等 `AssignmentManager` 就绪后发首条创建命令，带重试上限 |

## 2. 程序集

| asmdef | 路径 | 说明 |
|--------|------|------|
| `PlayModeTests` | `Tests/PlayMode/PlayModeTests.asmdef` | PlayMode UAT，`UNITY_INCLUDE_TESTS` 下启用 |
| `Tests.Editor` | `Tests/Editor/Tests.Editor.asmdef` | EditMode UAT，仅 Editor 平台 |

## 3. 使用注意

- 需场景内存在 `EventBusBootstrap`、`AssignmentManager` 与工厂/注册表等前置条件（见各类型 XML 注释）。
- 与正式剧情下达指令路径并行时，注意 `businessUid` / `spawnPolicy` 避免重复实例。
