# 实现：剧情进度（StoryProgress）

| 字段 | 值 |
|------|-----|
| 模块键 | StoryProgress |
| 主代码路径 | `Assets/01_Scripts/Runtime/Progress/StoryProgressService.cs` |
| 数据资产 | `Assets/04_Data/进度系统/StoryProgressCatalogSO` |

## 1. 职责边界

- **负责**：游戏进度 `story_progress` 字段读写、解锁 gates 查询（stub）。
- **不负责**：对话节点记忆（DialogSystem）、邮件已读（MessageSystem）。

## 2. 核心类型

| 类型 | 说明 |
|------|------|
| `StoryProgressService` | 进度 ID 集合运行时服务 |
| `StoryProgressCatalogSO` | 进度 ID 与说明对照表（占位） |
| `StoryProgressSaveData` | 存档字段（嵌入 `AssignmentSaveBundle` 后续扩展） |

## 3. 写入点（规划）

- 对话节点完成
- 关键邮件已读
- 主线节拍触发器
