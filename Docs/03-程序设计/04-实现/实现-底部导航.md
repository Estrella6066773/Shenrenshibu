# 实现：底部导航（UI_Navigation）

| 字段 | 值 |
|------|-----|
| 模块键 | UI_Navigation |
| 关联设计 | `需求-底部导航.md`、`需求-主菜单与场景流.md`、`需求-存档与进度.md` |
| 主代码路径 | `Assets/01_Scripts/UI/Navigation/BottomNavigationBarController.cs` |

## 1. 行为摘要

- Win 按钮：展开/收起主导航按钮组。
- **快速存档**：向活动存档集追加**手动**子存档（`SaveProgressOrchestrator.TrySaveActiveCollection`）。
- **管理存档**：游戏内提示回主菜单管理（完整 UI 在主菜单）。
- **返回主菜单**：`SceneFlowCoordinator.LoadMainMenu()`。
- 关机子菜单：退出游戏、返回主菜单。
- 设置：可绑定与主菜单共用的 `UI_设置面板` 预制体。

## 2. 与主菜单分工

| 功能 | 主菜单 | 底部导航 |
|------|:------:|:--------:|
| 继续/新游戏 | ✓ | — |
| 存档管理面板 | ✓ | 引导回主菜单 |
| 快速存档 | — | ✓ |
| 设置 | ✓ | ✓（可选） |
