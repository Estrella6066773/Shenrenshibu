| 字段 | 值 |

| 模块键 | UI_Navigation |
| 关联设计 | `需求-底部导航.md`、`需求-主菜单与场景流.md`、`需求-存档与进度.md`、`需求-界面与弹窗.md` |
| 主代码路径 | `Assets/01_Scripts/UI/Navigation/BottomNavigationBarController.cs` |

## 1. 行为摘要

- Win 按钮：展开/收起主导航按钮组。
- 子按钮：快速存档、管理存档、关机（含子菜单：退出、回主菜单）、设置。
- 设置子页（REQ-NAV-004）：音量、全屏/窗口、分辨率等；变更应持久化。
- 返回栈（REQ-NAV-005）：子页关闭回到展开主导航；消息 Tab 切换不强制收起 Win。
- 支持 `autoResolveNavRootIfMissing` 按子物体名称解析容器。
