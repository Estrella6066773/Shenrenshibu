# 实现：数据资产布局（DataAssets）

| 字段 | 值 |
|------|-----|
| 模块键 | DataAssets |
| 关联设计 | `需求-数据资产与目录约定.md`、`需求-简历标签与数据规则.md` |
| 主资源路径 | `Assets/04_Data/` |

## 1. 运行时内容目录（仓库现状摘要）

| 区域 | 路径模式 | 用途 |
|------|-----------|------|
| 对话与剧情 | `04_Data/对话dat/频道配置/`、`角色库/`、`剧情演出/` | `ChatChannelRegistry*`、`ChatRoleDatabase*`、`ChatStory*` 等 |
| 消息系统 | `04_Data/MessageSystem/` 等 | 邮件相关数据库（以 `ShenRenShiBuAssetMenu` 常量为准） |
| 标签 | `04_Data/标签系统/标签类型/*` | 多子类（性格、义体、状态…）`EmployeeTagSO` |
| 简历 | `04_Data/简历系统/Templates/` | `EnhancedResumeTemplateSO` |
| 委托 | `04_Data/委托系统/` | `AssignmentTemplateSO`、`AssignmentBranchTreeSO` |
| 证件照 | `04_Data/证件照/` | `PortraitConfigSO` |

## 2. 编辑器创建路径（与代码常量对齐）

`ShenRenShiBuAssetMenu` 中：

- 通用 `BasePath` 已统一为 **`Assets/04_Data/`**。
- 标签、简历、委托、证件照、对话与邮件创建入口均指向 `04_Data` 下子路径。

字段级说明见 [数据字典](../03-数据字典/README.md)。

## 3. 关联模块

- Employee / MessageSystem / Assignment 的 IMPL 中 SO 类型均依赖此处资产。
