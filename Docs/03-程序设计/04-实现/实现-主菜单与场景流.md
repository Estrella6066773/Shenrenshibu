# 实现：主菜单与场景流（MainMenu / SceneFlow）

| 字段 | 值 |
|------|-----|
| 模块键 | MainMenu / SceneFlow |
| 主代码路径 | `Assets/01_Scripts/Presentation/MainMenu/`、`Assets/01_Scripts/Runtime/SceneFlow/` |
| 程序集 | `ShenRenShibu.Presentation`、`ShenRenShibu.Runtime` |
| 最后对照 | 2026-07-02 |

## 1. 职责边界

- **负责**：独立主菜单场景、继续/新游戏、存档管理/设置入口、统一场景切换、Build Settings 首场景约定。
- **不负责**：玩法模块规则、游戏内底部导航业务逻辑。

## 2. 入口与生命周期

| 场景 | 启动链 |
|------|--------|
| `主菜单` | `MainMenuBootstrap` → `CampaignSaveCollectionService`（DDOL）→ `MainMenuController` |
| `主场景` | `RuntimeBootstrap` → `CampaignLoadOnStart` 读活动集主存档 → `StartupGateCoordinator` 门控 |

Editor：`神人事部/场景/安装主菜单场景`、`同步 Build Settings 场景列表`、`校验场景配置`。

## 3. 核心类型与文件

| 类型 | 文件 | 说明 |
|------|------|------|
| `GameSceneIds` | `Runtime/SceneFlow/GameSceneIds.cs` | 场景名常量 |
| `SceneFlowCoordinator` | `Runtime/SceneFlow/SceneFlowCoordinator.cs` | 场景切换 |
| `MainMenuController` | `Presentation/MainMenu/MainMenuController.cs` | 主菜单按钮逻辑 |
| `MainMenuBootstrap` | `Runtime/Bootstrap/MainMenuBootstrap.cs` | 主菜单存档集服务引导 |
| `IAboutPanelHost` | `Presentation/MainMenu/IAboutPanelHost.cs` | 关于接口（首版占位） |
| `SceneFlowSetupMenu` | `Editor/SceneFlowSetupMenu.cs` | 场景与预制体安装 |
