## 维护规则

1. 新增 SO 字段时，先更新本目录，再更新对应 `04-实现` 集成说明。
2. 影响策划填写的数据必须写清默认值、必填性与引用关系。
3. 运行时只读配置与运行时状态分表记录，不把存档字段混入 SO 字段表。
