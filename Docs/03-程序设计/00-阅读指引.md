# 程序与交付阅读指引

| 字段 | 内容 |
|------|------|
| 最后更新 | 2026-06-15 |

本指引按**内容类型**概述 **03-程序设计/** 内的文档：架构、实现、引擎映射与交付验收。**不按读者身份分类。**

全库入口见 [../00-阅读指引.md](../00-阅读指引.md)。玩法规则见 [02-系统设计/](../02-系统设计/README.md)；设定母本见 [04-设定/](../04-设定/README.md)。

## 一、本库回答什么问题

程序设计描述**数据结构、模块边界、判定逻辑、数据路径与引擎映射意向**。文首须链到关联的 `需求-*`；**不展开**大段玩法叙事（应链到系统设计或设定设计）。

工作区总览另见 [00-规范/总览-程序文档工作区.md](00-规范/总览-程序文档工作区.md)。

## 三、实现文档怎么读

每篇 `实现-<模块>.md` 通常包含：

| 章节 | 含义 |
|------|------|
| 元数据 | 模块键、`关联设计` 链到需求 |
| 数据与资产 | 表、枚举、ScriptableObject 路径 |
| 运行时行为 | 判定顺序、事件、模块间调用 |
| 与需求的偏差 | 若有，应同步 [待细化追踪](../00-规范/待细化追踪.md) |

模块总表见 [实现总览.md](04-实现/实现总览.md)。
