# 街区势力关系简表

## 第一条　表格用途

本表用**一句话定位**概括人事部与主要灰色势力的结构关系，供策划与文案快速对齐口径，避免同一版本中出现彼此矛盾的前提。

## 第二条　关系定位

| 势力 | 与人事部（**人类事务管理部**，[ORG-HRDEPT](../10-百科/组织/赫利俄斯/ORG-HRDEPT-人类事务管理部.md)）的结构关系 |
|------|--------------------------------------|
| “狼群” | 流亡家族组成的联合体，谋求地盘与秩序合法化，较易形成长期外包合作，也可能因责任切割与扶持次序发生变化。 |
| 鬣狗帮 | 对一批机会主义团伙的统称，专在乱局窗口抬价套利，波动大、稳定性低；一旦合作，往往伴随高勒索与反噬风险。 |
| 花豹党 | 强调社区自保与延续，依托地理与人脉门槛，与人事部既有合作空间，也存在摩擦。**叙事当下**：高层与 **[自由人联合阵线](../10-百科/组织/ORG-MOV-FUF-自由人联合阵线.md)** **暗中来往**；组织体整体与联合阵线**接触有限** → **未惊动企业**（非沦陷区式武装反对）。 |
| 反抗军 | **设定待商榷**（[ORG-REBEL](../10-百科/组织/ORG-REBEL-反抗军.md) 条目标 `draft`）。叙事当下**无单一武装反对主力**；张力来自帮派、**非实控区**与花豹—联合阵线暗线等组合。 |
| 自由人联合阵线 | 跨城武装—政治网络的一支；在本市曾与企业在沦陷区拉锯，主力撤出后仍有残余势力在联合州北部（母本见 [ORG-MOV-FUF](../10-百科/组织/ORG-MOV-FUF-自由人联合阵线.md)）；人事部外宣常与叛乱／废墟叙事捆绑。 |
| 联结会 | 宗教性质的结社，信奉联结之神，以兄弟姐妹互称；与 **旧人事部** 存在一定历史关联，与 **重组后人事部表面上无往来**；内部档案定性为 **危险宗教**，但因第九区萨满教、造主教及其分支等宗教生态庞杂而 **难以妥善管辖**；成员与 **艾萨克** 私下联络，并向废墟区 **残留智能** 提供 **非军事级** 援助（母本见 [ORG-MOV-BRIDGECULT](../10-百科/组织/ORG-MOV-BRIDGECULT-联结会.md)）。 |
| 人事部 | 制度目标是完成委托并控制损失；原则上可以与各势力做交易，也可能在审计与舆情压力下面临归责。 |
