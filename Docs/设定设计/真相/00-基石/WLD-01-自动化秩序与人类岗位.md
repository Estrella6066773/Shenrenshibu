# 自动化秩序与人类岗位

## 第一条　分工总貌

在本社会中，绝大多数**可规范化、可流程化**的劳动已由人工智能与机器承担。人类岗位整体上收缩到**不确定性高、风险高、责任难以向外转移**的区间；这是真实的社会现象——人们因岗位被机器替代，逐渐转向服务业或陷入失业。

世界基调为**资源不足且分配失衡**：联合州私有制下，财产多归个人；对企业而言，**投几个低端工厂让底层勉强维生**，往往比系统性修缮民生与社区治安**更有性价比**。赫利俄斯集团专门设立若干容错率高的**低端产业**——成本极低、对公司几乎无经济收益，主要意义是让失业人口有工可做、有途径换取基本消费品，从而稳定辖区人口与**考核指标／舆情底线**（避免暴动成本失控）。即便如此，下游管理者仍层层盘剥，**高技术低生活**的底层社会长期存在。
