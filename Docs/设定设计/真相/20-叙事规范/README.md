# 叙事规范

本目录约束**对外叙事**与**玩家能直接感受到的体验尺度**：基调、母题、终局走向——**不是**具体剧情细纲。剧情实例、任务文本与脚本放在 [叙事生产稿/](../../叙事生产稿/README.md) 或版本仓库；如有冲突，以本目录的硬性约束为准。

设定 `id` 前缀 **`NAR-*`**（*Narrative norms*）登记于 [管理规则/id-registry.yaml](../../管理规则/id-registry.yaml)。
