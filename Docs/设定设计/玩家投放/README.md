# 对玩家冻结层（玩家投放）

**阅读指引：** 对外无剧透入口见 [v0.1 世界观导览](v0.1/对外读本/00-世界观导览与阅读分层.md)；完整剧情阅读路径见 [../00-剧情阅读指引.md](../00-剧情阅读指引.md)。

本目录收录**已通过审批、可向指定范围的玩家或公众披露**的文案与导览。其内容应为 `真相/` 的审慎子集，并按**产品版本**分目录冻结。

## 当前发布版本

**[v0.2](v0.2/README.md)**（2026-06-13）：v0.1 全量继承 + 四科科长 `CHR-*` P0 人物导览。

## 版本目录规则

- 每一子版本（如 `v0.2`）要包含：`README.md`、`CHANGELOG.md`、`manifest.yaml` 与 `对外读本/`。
- 历史版本（如 [v0.1](v0.1/README.md)）保留供溯源；升版时更新本页**当前发布版本**指针。
- `manifest.yaml` 列明本版本对外生效的设定 `id` 集合；增删要伴随 `CHANGELOG` 记录。
- 未经审批不得将 `visibility: internal` 的条文整段复制到 `对外读本/`。
