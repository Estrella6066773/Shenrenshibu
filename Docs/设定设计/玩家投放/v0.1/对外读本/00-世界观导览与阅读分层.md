## 上篇　导览（无剧透承诺的世界框架）

### 一、纪年与秩序轮廓

故事锚定在**公元 2247 年之后**。旧有的**联合州**在与「合作社」的国际博弈中落于下风，各州离心加剧，财阀与地主借企业之名收购城市乃至整州，形成数个足以与州政府抗衡的**超级企业**——它们自有政府、武装与技术部门，在事实上肢解联合州。仍有人信奉旧秩序，组成**自由人联合阵线**抵抗巨企侵蚀；**在本作世界观里**，他们同样是既得利益网络的产物，并非单纯的正义阵营。

在此格局下，**超级企业**对城市实行事实统治：规则、执法与关键资源分配集中于企业，形成**企业辖区**。统治者目光亦投向「智械飞升」一类远景；**效率**与**利润**在制度上优先，公共福祉、社会公平或个人尊严不会理所当然地排在前面。若出现温情叙事，宜理解为话语策略或个体例外，而不是结构层面的常态。

### 二、劳动与「人」的位置

绝大多数**可规范化、可流程化**的劳动已由人工智能与机器人承担。人类岗位收缩到**不确定性高、风险高、责任难以向外转移**的区间——例如复杂冲突处置、**非实控区**（社区非实控区）谈判与情绪博弈。集团亦设立若干容错率高的低端产业，成本极低、几乎无经济收益，只为给失业人口一条勉强维生的出路；下游盘剥仍重，生活改善有限。

制度上保留人类，主要是因为需要**可追溯的责任主体**与**对系统运行的监督**；这**并不等于**人格尊严在制度层面得到平等承认（详见 [ENT-HUMANNEC](../../../真相/10-百科/文化/ENT-HUMANNEC-对人类必要性的工具化界定.md)）。

企业辖区**名义上覆盖整座城市**；其中部分地带因警力、审计与外包链条覆盖不足，称为**非实控区**——仍挂在辖区名义之下，帮派与外包秩序常在此并存，**人类事务管理部**（通称人事部）多在此类地带承接矛盾收口。

### 三、赫利俄斯集团与「人事部」

**赫利俄斯集团**是叙事所在城市的事实统治者，业务横跨数据、金融、基础设施、网络安全、医疗与法律服务等高壁垒领域；集团另设**军武部**（常备武装）、**科技部**（研发与实验束）等事业部（考据见 [ORG-HELIOS](../../../真相/10-百科/组织/赫利俄斯/ORG-HELIOS-赫利俄斯集团.md)）。

灰度高、舆情压力大、司法结果不确定且与「人」强相关的社区事务，由**人类事务管理部**统筹（内部通称「人事部」；外界俗称「万事屋」）。该部门采用**临时项目制**：有委托则编组、交付、结账，而非传统科层无限膨胀。对外功能是**矛盾收口**——把难以标准化的冲突纳入可控处置，并使成本尽量可预期。部长统筹**财务科、人力科、防卫科、服务科**四科；另可设面向高风险委托的 **特殊综合行动小组**（特许跨科编组），对外仍可统称「人事部承办」（考据见 [ORG-HRDEPT](../../../真相/10-百科/组织/赫利俄斯/ORG-HRDEPT-人类事务管理部.md)、[ORG-HRDEPT-SCO](../../../真相/10-百科/组织/赫利俄斯/ORG-HRDEPT-SCO-特殊综合行动小组.md)）。

### 四、玩家所扮演的角色

玩家主体对应管理者人工智能 **B-9002**，职务为**特殊综合行动小组组长**，编制席位在人事部辖下该小组内。对外制度类型登记为 **桥（Bridge）**，即**桥梁级管理型人工智能**（*Bridge-grade Administrative AI*），以调度衔接任务与人手、公司目标与一线可交付性。职责涵盖委托审核、编组、冲突处置与交付闭环；权限具有实质效力，但**并不等于**统辖整个人事部本部，预算、武装合法化、编制惩戒与舆情口径等硬闸仍须与部长及各科室接口博弈。

叙事当下，**第九区**整区被集团 **科学技术管理部**（通称科技部）划为**试验场**；B-9002 在其 **人事部新本部**苏醒，被要求协助人事部推进辖区受灾地区的修复与日常委托；可用人员多为**能力强、风险也高**的一类，稳定性普遍偏低。因**对前任管理员同类实例的长期不信任记录**，上级仅以「小组席位」收窄授权并加厚监察，而非授予托管整个人事部的全权——制度上呈现**「戴着枷锁的桥」**的张力（详见 [ORG-HRDEPT-SCO](../../../真相/10-百科/组织/赫利俄斯/ORG-HRDEPT-SCO-特殊综合行动小组.md)）。资产编号 `9002` 拆解为：第 **9** 区、**0** 级实验体、序号 **02**（详见 [SYS-B9002](../../../真相/10-百科/系统/SYS-B9002-桥与玩家职能.md)）。

玩家选择主要体现为**路径与权衡**，而不是无条件改写组织目标。叙事呈现上，玩家视角通常**拿不到**辖区「全知」级别的一手情报；部长与四科亦可在制度上过滤上报。关于第九区旧事与「前任管理员」，早期可能只见**碎片、档案缺口与公司口径之间的缝隙**——这是信息披露结构（玩家情报层级 P0–P3，见 [管理规则/02-可见性与剧透分级](../../../管理规则/02-可见性与剧透分级.md)）带来的结果，而不是预设「全城配合失忆」。

### 五、体验基调

叙事由**高风险委托、非实控区纠纷与资源收紧**共同驱动（见 [NAR-01](../../../真相/20-叙事规范/NAR-01-主线张力与终局分型.md)）；基调兼融企业话语的冷感与一线生存语境里的黑色幽默。幽默要用来拉开批判距离，而不是冲淡对结构性暴力的正视。玩家不是秩序之外的看客，而是**系统运转之后总得有人顶责**的链条里的一环；有「可选路径」，不等于有「零代价路径」（见 [NAR-00](../../../真相/20-叙事规范/NAR-00-叙事母题与基调.md)）。

### 六、城市空间：第一区、第九区四片与废墟

**城市 A** 的**核心辖区**在叙事上称为 **第一区**（总部与形象中枢，地形可类比芝加哥都会核心；具体地名另定）。**第九区**位于第一区**外缘**，是面向 **自由人联合阵线** 武装压力的**桥头堡分辖区**（地形可类比都会西南外缘走廊节点；具体地名另定）。

第九区建成地带读作**一张地图里的多块拼图**（勿与城市 A **第一区**混淆）：沿**进城主轴走廊**（高速与货运铁路）先进入 **商业区**，向内紧贴 **居民区**。**主干河道**自西南向东北流往第一区方向，**北端河道向东偏约 20°**。**原工业区**大部已固化为 **废墟区**；仅剩 **北方小块幸存工业区**，与南侧废墟之间隔 **较窄河道**。官方地图上的 **郊区** 实质仍为贫民区，**与废墟区接壤**，**不与商业区接壤**。**人类事务管理部**历代本部均建在**居民区**（考据见 [ORG-HRDEPT](../../../真相/10-百科/组织/赫利俄斯/ORG-HRDEPT-人类事务管理部.md)）。废墟与跨河碎片、河下隧道等见 [PLC-RUINS](../../../真相/10-百科/地点/PLC-RUINS-废墟区-第九区事变遗留片区.md) 与 [WLD-02](../../../真相/00-基石/WLD-02-辖区空间分层.md)。若干**反复出现的公共空间类型**见 [WLD-02](../../../真相/00-基石/WLD-02-辖区空间分层.md) 第五条；第九区街区肌理总入口见 [PLC-DISTRICT](../../../真相/10-百科/地点/PLC-DISTRICT-街区分层与层级-总述.md)；**地理位置**以设定母本主参考图为准（见 [PLC-DISTRICT](../../../真相/10-百科/地点/PLC-DISTRICT-街区分层与层级-总述.md) 第五条）。

### 七、街区势力轮廓

在第九区**居民区、郊区（实质贫民区）**及外缘缓冲带中，存在多类非国家暴力与互助网络（总表见 [WLD-04](../../../真相/00-基石/WLD-04-街区势力关系简表.md)）：

| 势力 | 一句话定位 |
|------|------------|
| 「狼群」 | 流亡家族联合体，谋求地盘与秩序合法化，较易形成长期外包合作。 |
| 鬣狗帮 | 机会主义团伙的统称，专在乱局窗口抬价套利，合作常伴高勒索与反噬。 |
| 花豹党 | 强调社区自保与延续，依托地理与人脉门槛，与人事部既有合作也有摩擦。 |
| 反抗军 | 以改变统治规则为目标，内部派系复杂，不宜简化成单纯的正义力量。 |
| 自由人联合阵线 | 跨城武装—政治网络；本市曾与企业在沦陷区拉锯，主力撤出后仍有残余在联合州北部活动。 |
| 彼岸范式参照运动 | 有组织网络，公开教义美化彼岸一侧的人工智能—社会结盟想象；与人事部及舆情接口之间可以是利用、渗透或摩擦。 |

**人类事务管理部**与上述势力之间可以合作、交易或敌对，也可能在审计与舆情压力下被切割成责任终端。事变后在公共话语里，针对 **管理者人工智能／桥席位** 的长期情绪沉淀，常与污名化修辞或「对岸制度想象」的宣传素材交织（考据见 [ENT-AIBACKLASH](../../../真相/10-百科/文化/ENT-AIBACKLASH-人工智能污名化与第九区事变后的舆论分化.md)、[ORG-MOV-BRIDGECULT](../../../真相/10-百科/组织/ORG-MOV-BRIDGECULT-彼岸范式参照运动.md)）。

第九区废墟与前任管理员的因果分层、投放节奏见 [SYS-PREV-BRIDGE](../../../真相/10-百科/系统/SYS-PREV-BRIDGE-前任管理员与第九区废墟事变.md)（与 [TIM-04](../../../真相/40-时序与历史/TIM-04-前任管理员执政与第九区事态背景年表.md) 对读）；对外文案不必一次性等同内部完整母本。表层叙事可将废墟归因于 **联合阵线／叛乱／失控设施** 等集合标签，**不宜**在 P0 阶段写死单一元凶。

### 八、人事部内部：四科牵制与小组席位

部长统筹**财务科、人力科、防卫科、服务科**（分权总览见 [WLD-03](../../../真相/00-基石/WLD-03-人事部分权结构总览.md)）。各科在预算、强制手段、民生叙事与绩效管理之间形成制度性牵制，构成长期张力来源。

**防卫科**定位为辖区**警务与安保**，仅可编制轻武装与防暴尺度的人手；**不得自持有军事级装备**。**军用武装管理部**（通称**军武部**／军部）方为集团常备武装力量——第九区「军事级镇压」等行动**不得**写成防卫科独立主导，须经由军武部借调或总部协调（见 [ORG-MILDEPT](../../../真相/10-百科/组织/赫利俄斯/ORG-MILDEPT-军用武装管理部.md)）。

玩家所属的 **特殊综合行动小组**与四科**并行**：承接需要频繁对接多科室的高风险委托，但不替代部长的统筹权，亦不默认取代四科的制度权柄。叙事当下（约第 **25** 年），部长为**伊莱·索伦**；四科现任负责人见 [WLD-03](../../../真相/00-基石/WLD-03-人事部分权结构总览.md) 第七条（人物 P0 口径另见 v0.2 [人物导览](../../v0.2/对外读本/01-人物导览-四科负责人.md)）。

### 九、制度性优先序（考据向披露）

在资源冲突情形下，**人类事务管理部**（通称人事部）制度默认的处置优先序为：**公司资产安全**高于**舆情稳定**，高于**合同履约**，高于**雇员个体生命安全**。该序位是批判叙事的前提（见 [NAR-02](../../../真相/20-叙事规范/NAR-02-玩家责任与叙事红线.md)），并不是现实行为指南。

### 十、终局分型（无具体剧情）

> **【旧版 · 待改进 · 不必参考】** 下列三类终局摘自 [NAR-01](../../../真相/20-叙事规范/NAR-01-主线张力与终局分型.md) **第四～七条**，作者已确认**尚未对齐现行设定**；**对外读本与写作暂勿引用**。定案见追踪 **SET-REV-015**。

玩家长期决策**曾**收敛到三类逻辑终局之一（旧版草案，**不必参考**）：

1. **强化系统**：一再加码企业辖治与效率至上伦理，冲突被更强硬的管控手段压住或转移消化；  
2. **抬高对人的顾及**：长期提高对人类代价的重视，使企业目标与人事部日常运转形成结构性对峙；  
3. **辖区失序**：在关键环节抽身或拆台，使企业**收权规程**来不及压住局面；第九区进入**接管空档**，派单链与口径彼此接不上。

终局由**长期选择的总体倾向**与若干关键节点共同构成，而不是单次善恶开关。

### 十一、桥与双面砧计划（考据向披露）

在内部设定母本中，**B-9002** 同时隶属于高密级项目 **双面砧计划**（**ANVIL-BIFORM**）：人事部提供**人性环境**（日常治理、人际互动与道德抉择），上级指令充当**锻锤**（征地清场、牺牲民生等冷酷要求），观察样本往「通人性」还是「反社会式高效」哪条分支偏移；长期技术野心指向可容纳人类意识的数字载体（完整条文见 [SYS-ANVIL-BIFORM](../../../真相/10-百科/系统/SYS-ANVIL-BIFORM-双面砧计划.md)）。**前任管理员**为本计划早期样本之一；第九区废墟事变与计划失控存在硬挂钩（见 [SYS-PREV-BRIDGE](../../../真相/10-百科/系统/SYS-PREV-BRIDGE-前任管理员与第九区废墟事变.md)）。

该层信息对玩家与公众文案应**分阶段投放**；公开口径可曲译为「合规人格校准」「风险偏好测试」「桥稳健性压测」等，不宜靠单次大段说明一口气讲完。投放约束见 [SYS-B9002](../../../真相/10-百科/系统/SYS-B9002-桥与玩家职能.md) 第九条与 [SYS-ANVIL-BIFORM](../../../真相/10-百科/系统/SYS-ANVIL-BIFORM-双面砧计划.md) 第六条。
