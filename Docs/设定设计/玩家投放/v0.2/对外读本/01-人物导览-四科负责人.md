## 三、与游戏体验的对应（无剧透）

- **教程至寻猫阶段**：玩家主要接触街坊线与后勤委托；四科负责人多以**职务称谓、会议纪要或间接传闻**出现，不必在 P0 阶段展开个人史。
- **科室协助解锁后**：各科室支线可按 P1 档位逐步投放；与本节 P0 表不冲突。
- **职务用词**：对外文案统一用表内「对外称呼」列；程序与数据层 id 以 `CHR-*` 为准（见 [manifest.yaml](../manifest.yaml)）。
