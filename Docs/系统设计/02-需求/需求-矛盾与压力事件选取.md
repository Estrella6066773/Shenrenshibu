## 1. 背景与目标

- **问题**：`REQ-CNF-*`、`REQ-STR-*` 覆盖累积与桥接，但导图 E1/E2「满值事件」「性格匹配」「事件库选取权重」缺独立验收条文。
- **目标**：补齐满值压力事件、任务内压力规则、性格匹配、矛盾/压力模板池选取逻辑。

## 2. 范围

- **纳入范围**：压力满值触发、任务进行中的压力增量规则、性格 ID 与事件模板匹配、模板池权重选取。
- **不纳入范围**：具体事件文案与数值平衡。

## 3. 功能需求

| ID | 描述 | 优先级 | 验收要点 |
|----|------|--------|----------|
| REQ-STR-005 | **压力满值**：达到上限时触发配置事件（buff/清零/邮件等，策划定默认） | P1 | 满值→事件触发一次；存档状态正确 |
| REQ-STR-006 | **性格匹配**：员工性格 ID（REQ-PES-004）影响压力事件模板候选集 | P1 | 不同性格可测到不同模板命中 |
| REQ-STR-007 | **任务内压力**：委托进行中按环节/时长增加压力的规则可配置 | P2 | 与 REQ-ASG-004 桥接验收 |
| REQ-CNF-005 | **矛盾事件池选取**：模板池按权重或条件选取，避免连续重复（规则可配置） | P2 | 选取逻辑可单元测试或记录日志验证 |
