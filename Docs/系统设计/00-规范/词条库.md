# 词条库（玩法标准中文名）

| 字段 | 内容 |
|------|------|
| 状态 | 草案 |
| 最后更新 | 2026-06-15 |

本文统一**系统设计**与**程序设计**文档中的中文术语，避免同一概念多种叫法。叙事专有名词另见设定库 `06-名词与术语`（待建）或 StoryLib 治理规则。

## 核心循环

| 标准名 | 英文模块键 | 说明 |
|--------|------------|------|
| 核心消息 / 启动门控 | CoreMessaging | 全局总线、`RuntimeInitializationCompletedEvent` 等 |
| 玩家经济 / 日历时钟 | PlayerEconomy | 资金、日结、任务结算 |
| 员工 | Employee | 员工数据、状态、标签 Buff |
| 委托 | Assignment | 任务创建、Tick、分支、队伍交互 |
| 矛盾 | Conflict | 矛盾累积、协调、邮件桥接 |
| 压力 | Stress | 压力分段、七宗罪事件 |
| 消息系统 | MessageSystem | 邮件、聊天、日志 UI |
| 对话播放 | DialogSystem | 剧情节点与选项（命名空间 `DialogSystem.*`） |
| 简历窗口 | Resume | 招聘 UI、排序、插卡 |
| 证件照 | Portrait | 分层立绘生成 |
| 待命区 | IdleArea | 收纳与整理 |
| 底部导航 | UI_Navigation | 全局页壳 |
| 数据资产 | DataAssets | `04_Data` 目录与 SO 约定 |

## 设定叙事专名（与写作黑名单区分）

| 专名 | 含义 | 登记 |
|------|------|------|
| **非实控区** | 名义上全城皆为管控区域；实际管控薄弱、帮派与外包秩序并存的地带 | [管理规则/01 §8](../../设定设计/管理规则/01-命名文件与ID规则.md) |

| 废止 | 现用 |
|------|------|
| **灰区**（设定专名） | **非实控区** / **社区非实控区** |
| **非正式辖区**（过渡用语） | **非实控区** |
| **叙事合同** | **叙事规范** |

工程文档中的含糊「灰区」仍禁用；须用待对齐项、部分符合等具体说法（见 `.cursor/rules/no-coined-terms.mdc`，本地保留、不进 Git）。
