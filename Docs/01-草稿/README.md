# 草稿

本目录用于存放《神人事部》的灵感、脑暴、全局分解图与快速对照稿。

## 使用原则

- **不追求完美**：允许凌乱、跳跃；正式收束后迁入 `02-系统设计/` 或归档。
- **快速记录**：想到就写，不必等结构完整。
- **定期收敛**：有价值的内容整理进正式文档；本目录详情图随 `02-系统设计/` 变更同步更新。
