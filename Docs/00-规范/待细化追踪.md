## 使用说明

### 收录范围

任一库内尚未定案、未实现、文档与代码**有偏差**、或验收未勾选的项。

### 编号前缀

| 前缀 | 库 |
|------|-----|
| `GOV-` | 跨库治理 |
| `SYS-` | 系统设计（`02-系统设计/`） |
| `IMPL-` | 程序设计（`03-程序设计/`、`Assets/`） |
| `UAT-` | 交付验收（冒烟 / 自动化） |
| `SET-` | 设定设计（`04-设定/`） |

### 状态

| 状态 | 含义 |
|------|------|
| **开放** | 未开始或缺文档 |
| **draft** | 需求/母本草稿，不可作验收依据 |
| **partial** | 主路径存在，规则或集成未闭环 |
| **已定案·待验收** | 决策或代码已有，REQ/UAT 未同步 |
| **策划待定** | 阻塞程序的规则表、枚举待协作产出 |
| **旧版·不必参考** | 保留追溯，勿在新稿中引用 |

### 优先级与负责

| 级 | 含义 | 典型负责 |
|----|------|----------|
| **P0** | 阻塞提交、一致性或对外演示 | 开发者 |
| **P1** | 当前竖切迭代 | 开发者 / 协作 |
| **P2** | 排队 | AI / 人工 / 协作 |
| **P3** | 可选 | 协作 |
