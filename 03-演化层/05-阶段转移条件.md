# 阶段转移条件

> [参考] 本词条当前为设计参考，阈值与触发条件可在玩法联调后调整。

本页用于定义阶段切换的硬条件，确保演化节奏由状态变化驱动，而非由剧情宣告驱动。

## 转移规则

本段意义：给出阶段跃迁的最低触发框架，便于策划在联调时替换为具体数值阈值。

- [参考] A -> B：基础设施恢复率与治安稳定率达到最低线。
- [参考] B -> C：资源紧缩与冲突事件连续超阈值。
- [参考] C -> D：玩家获得跨科室结构性权限并触发上级审查。

## 使用边界

本段意义：限定使用规则，防止为了推进剧情跳过系统状态检验。

- [参考] 阶段转移必须满足条件，不允许仅靠剧情宣告。
