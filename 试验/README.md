# 试验：文档库结构（正式版草案）

本目录为《神人十步》世界观文档库的**目标架构与操作规范**试验场。文本书写采用正式说明体例，供内部协作与对外素材管线对齐使用。

## 定位

- **本目录不替代**仓库内既有 `正文/` 中的词条内容；当前阶段以「规范 + 索引 + 对照」为主，词条内容仍以 `正文/` 为单一事实来源。
- **本目录替代**的是「文件夹怎么分、先读谁、谁不能改、对外从哪出」的组织方式；待评审通过后，可将 `正文/` 按本结构整体迁移，迁移时沿用既有 `LORE-*` 标识，不重新编号。

## 顶层结构

| 路径 | 职责 |
|------|------|
| `试验/docs/00-入职与导读/` | 新人上手顺序、协作禁令、与 `正文/` 的临时关系说明 |
| `试验/docs/10-术语与命名/` | 术语主表、文件命名与路径约定（中文可读） |
| `试验/docs/20-设定真值/` | 对象五域 + 总索引 + **与 `正文/` 的词条对照表** |
| `试验/docs/30-叙事资产/` | 任务弧、区域叙事、语气规范；**引用**设定真值，不新增公理 |
| `试验/docs/40-玩家投放包/` | 按版本与渠道组织的对外可读切片；与内部总览解耦 |
| `试验/docs/50-治理与工程/` | 元数据模板、ID 注册表、评审清单、可见性总约定、变更裁定 |

## 使用顺序（摘要）

1. 阅读 `docs/00-入职与导读/新人阅读路径.md`。
2. 查阅 `docs/20-设定真值/_总索引.md` 与 `词条来源对照表.md`，定位 `正文/` 中的真值宿主文件。
3. 修改设定时遵守 `docs/50-治理与工程/` 中的模板与评审顺序；对外文案只自 `docs/40-玩家投放包/` 出口或经其登记的渠道。

## 版本说明

- **文档库结构版本**：`试验-2026-05-04`
- **内容真值版本**：以各词条 YAML 中 `status` 及 `正文/` 内修订为准；本目录不单独维护剧情版本号。
