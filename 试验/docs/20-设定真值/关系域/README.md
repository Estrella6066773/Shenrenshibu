# 关系域

## 职责

回答「实体之间如何连接」：控制、依赖、冲突、交易、渗透等。每条关系词条**单一主类型**；复合关系拆条，用 `relates_to` 互指。

## 关系类型枚举

- `control`：支配与被支配  
- `dependency`：相互依存  
- `conflict`：对立与矛盾  
- `transaction`：交换与互惠  
- `infiltration`：渗透与影响  

类型代码写入词条元数据 `type` 字段，与正文小节标题保持一致。
