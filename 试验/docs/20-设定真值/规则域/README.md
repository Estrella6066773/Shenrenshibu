# 规则域

## 职责

回答「世界如何被约束并运行」：正式法规、半正式规章、灰区默契、处置程序等。每条规则须写明适用对象、空间、触发条件与执行者，并至少关联一条观测域或时序域后果，以便校验可玩性。

## 规则类型

- **正式规则**：法律、条例等具强制力的规范  
- **半正式规则**：组织内部约定、部门规章  
- **非正式规则**：街区共识、灰区默契  
- **处置规则**：处罚、问责、重置等执行性机制
