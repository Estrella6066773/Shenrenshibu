# 实体域

## 职责

回答「世界中存在哪些可独立指称的对象」：人物、组织、地点、资源、技术、事件等。每个对象对应**一条**主词条；复合对象须拆分为多条并通过关系域链接。

## 子类范围

| 子类 | 说明 | 现行正文位置 |
|------|------|----------------|
| 人物 | 个体角色、人格化管理体等 | `正文/01-实体域/人物/` |
| 组织 | 机构、派系、企业单元等 | `正文/01-实体域/组织/` |
| 地点 | 地理、场所、辖区网格等 | 待建；规划路径同域下 `地点/` |
| 资源 | 物资、资产、配额载体等 | 待建 |
| 技术 | 科技、技能、工程方法等 | 待建 |
| 事件 | 已发生或结构性的重大变故 | 待建；与 `时序域` 节点区分：事件词条侧重实体侧后果，时序词条侧重分期与触发 |

## 录入约束

- 实体名称在设定内须唯一；别名字段写入词条元数据，不在多个文件重复定义。
- 「谁能知晓」不写入实体域长文；知情权由内嵌 `[投放注记]` 与可见性矩阵约束。
