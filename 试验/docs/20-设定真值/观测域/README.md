# 观测域

## 职责

回答「外界能观察到什么」：指标、舆情、行为迹象、制度后果。**不发明**新的身份类别、法律或组织结构；只归档可由上游推导或证实的可见结果。

## 观测对象类型

- **quantitative metric**：可量化指标  
- **narrative sentiment**：舆论与叙事情感倾向  
- **behavioral signal**：行为模式信号  
- **institutional outcome**：制度性结果
