# 时序域

## 职责

回答「世界如何随时间变化」：历史节点、阶段状态、转移触发、持续变量等。每个节点须有前因后果，禁止孤立年份堆砌。

## 时序对象类型

- **milestone**：历史转折点  
- **phase**：阶段整体特征  
- **trigger**：状态转移触发条件  
- **persistent variable**：长期变量

## 录入约束

关键节点须至少关联一条实体变化与一条规则或阶段变化（在 `relates_to` 或正文中显式写出）；转移条件尽量量化或可判定，避免纯形容词描述。
