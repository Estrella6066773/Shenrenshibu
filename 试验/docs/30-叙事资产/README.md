# 叙事资产

## 定位

本目录承载**任务弧、区域叙事基调、语气与禁忌**等写作资产。其与 `20-设定真值` 的关系为：

- 设定真值回答「世界客观上是什么」；
- 叙事资产回答「在当前版本下如何讲、讲多少、谁先知道」。

叙事资产**不得**引入与 `LORE-RULE-001` 冲突的新公理；不得默示尚未在实体、时序或规则域登记的实体或机制。凡新名词须在 `试验/docs/10-术语与命名/主编术语表.md` 或对应域词条中落点。

## 子目录约定

| 子目录 | 用途 |
|--------|------|
| `任务与弧光/` | 主线、支线、版本活动的大纲；条目中只写标识引用（`LORE-*`），不写长设定 |
| `区域与氛围/` | 分区街景、声景、任务密度与情绪曲线；可引用 `LORE-TIME-003` 与观测现象 |
| `语气与呈现/` | 人称、禁用比喻、营销边界；须与 `LORE-RULE-002` 呈现节对齐 |

## 当前状态
