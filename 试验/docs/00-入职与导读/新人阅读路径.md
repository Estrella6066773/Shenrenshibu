# 新人阅读路径

本文约定新成员在**十五分钟内**建立正确心智模型所需的阅读顺序。阅读材料以本仓库 Markdown 文件为准。

## 第一步：结构而非剧情

阅读 `试验/README.md`，明确「试验目录管结构，`正文/` 管当前真值」的分工，避免在未迁移阶段重复编辑两处正文。

## 第二步：公理与术语口径

打开 `正文/03-规则域/01-终局约束与世界背景公理.md`（词条标识 `LORE-RULE-001`）。  
该文件承载终局类型、世界背景公理与术语约束，是后续一切条目的**置信度上限**来源。

## 第三步：内部导航与对外禁令

打开 `正文/03-规则域/02-世界观总览与叙事锚点.md`（`LORE-RULE-002`）。  
该文件用于跨域导航与叙事呈现边界；**不得**将其全文作为玩家可见设定原文投放。对外切片须走 `试验/docs/40-玩家投放包/` 所约定之出口。

## 第四步：世界史与城市史母线

按顺序浏览：

1. `正文/04-时序域/03-世界历史总线.md`（`LORE-TIME-002`）
2. `正文/04-时序域/02-城市历史专区.md`（`LORE-TIME-003`）
3. `正文/04-时序域/01-历史线_关键节点.md`（`LORE-TIME-001`）

城市级分期、三群体结构与本地关键节点，分别以 `TIME-003`、`TIME-001` 为宿主；全球分期以 `TIME-002` 为准，避免在总览条目中重复展开长史。

## 第五步：组织、关系与人物

阅读顺序建议为：集团上位体 → 人事部组织 → 狼群锚点 → 权力冲突链 → 关键人物与管理体。

对应文件与标识见 `试验/docs/20-设定真值/词条来源对照表.md` 第二节。

## 第六步：观测层与投放

阅读 `正文/05-观测域/` 下已登记条目（`LORE-OBS-001`、`LORE-OBS-002`），理解「现象层」与上游实体、关系、时序的溯源关系。  
再阅读 `试验/docs/50-治理与工程/可见性投放总约定.md` 与 `正文/90-可见性投放/00-索引.md`，掌握 `[投放注记]` 的写法与对外禁忌。

## 第七步：作者向参照（非硬设定）

打开 `正文/03-规则域/03-叙事隐喻与社会学参照.md`（`LORE-RULE-003`）。  
该条目的置信度与用途边界在元数据中已标明；**禁止**作为玩家设定原文或宣发口径的唯一依据。

- 标识占用情况：`试验/docs/50-治理与工程/ID注册表.md`
