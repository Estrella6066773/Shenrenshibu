# 《神人事部》文档仓库

本仓库（GitHub：`Estrella6066773/ShenrenshibuStoryLib`）存放《神人事部》的**唯一文档 Git 库**。

| 路径 | 内容 |
|------|------|
| [Docs/](Docs/README.md) | 系统设计、程序设计、设定设计 |
| [`.github/`](.github/workflows/) | 文档持续集成 |
| [`Scripts/`](Scripts/Get-DocsDate.ps1) | 文档日期脚本 |

**与 Unity 工程配合使用：** 将本仓库克隆到本地目录 `神人事部/`；文档在 `Docs/`。`Assets/`、`ProjectSettings/` 由 UVC 管理，不进 Git。

```bash
git clone git@github.com:Estrella6066773/ShenrenshibuStoryLib.git 神人事部
```

**阅读入口：** 首次参阅请先打开 [Docs/00-阅读指引.md](Docs/00-阅读指引.md)；目录概览见 [Docs/README.md](Docs/README.md)。
