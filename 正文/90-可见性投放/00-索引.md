# 可见性投放索引

本目录仅作为"信息可见性"的导航索引，不承载独立的事实正文

## 使用方式

- 可见性规则写入各事实词条的正文段落，并以内嵌 `投放注记` 的形式呈现
- 本目录只提供跳转入口与注记模板

## 客户端 / 外包 / 宣发出口约定

- **`LORE-RULE-002`（世界观总览）**：默认**不进入**玩家可见资源包；YAML 中 `visibility.player: none`。对外叙事切片应来自任务、档案、观测词条与角色台词，而非总览全文。
- **`LORE-RULE-003`（叙事隐喻）**：**禁止**作为玩家设定原文投放；仅供作者与策划内部参照。
- **`LORE-OBS-*`（观测域）**：可作为「现象层」文案与 UI 的优先采信来源，须反向溯源至时序/实体/关系词条，观测域**不发明新设定**。

## 规则入口（按领域）

- `03-规则域/01-终局约束与世界背景公理.md`（术语口径与来源标注）
- `03-规则域/02-世界观总览与叙事锚点.md`（内部导航与禁令；非玩家投放主源）
- `04-时序域/01-历史线_关键节点.md`（历史节点的投放约束）
- `04-时序域/02-城市历史专区.md`（`LORE-TIME-003`，城市历史中的口径与群体叙述边界）

## 投放注记模板（通用）

```md
[投放注记]
- audience: 作者 | 角色(阵营/岗位) | 玩家 | 系统
- window: 何时可见（阶段/事件/任务后）
- channel: 载体（档案/台词/新闻/任务反馈）
- distortion: 无 | 轻度 | 显著（若有，写偏差方向）
```
