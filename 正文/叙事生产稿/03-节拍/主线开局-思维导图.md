## 便签（实现备注）

| 文本 | 用途 |
|------|------|
| 你的杨婶 | 寻猫线街区 NPC 称谓（与摩恩引荐相关） |
| 0 点不让放入职位，对所有任务有效 | 零编制不可派遣，全局防死锁 |
| 此处包括指引玩家让员工休整 | 与序 9 教程同条 |
| 需要富文本高亮某些关键词 | 霍尔特引荐等教程文案格式 |
