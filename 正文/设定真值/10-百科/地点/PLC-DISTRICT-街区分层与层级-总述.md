# 街区分层与层级（总述）

## 第一条　条文性质

本条为第九区内部空间分层的**总入口**，与基石 [WLD-02](../../00-基石/WLD-02-辖区空间分层.md) 对齐。辖区内反复出现的公共空间**类型纲要**（不是已定地名）见同条第三条；也可以查阅 [百科目录](../README.md)。

说明：A/B/C 分层是**人类事务管理部**（通称人事部）内部按阶层与掌控力度划分的工作分层，**不是**官方行政区划意义上的「街区」。

## 第二条　三层模型摘要

第九区具体地名尚在设计中；稿件中出现的「A 区 / B 区 / C 区」仅为**区分层次的临时称谓**，不等于正式地名。

| 层级 | 名称 | 功能摘要 |
|------|------|----------|
| A | 明亮层 | 监控密集、执法显眼，企业形象橱窗。 |
| B | 缝隙层 | 居住与灰产交织，委托与冲突的主战场。 |
| C | 暗面层 | 系统覆盖不全，黑市与隐匿叙事集中区域。 |

**已定名的事变遗留片区**：**废墟区**（见 [PLC-RUINS](PLC-RUINS-废墟区-第九区事变遗留片区.md)）叙事上主要落在 C 层，与 B 层边缘存在非法或半废弃的连通口。

## 第三条　势力与层级大致对应（可重叠）

- “狼群”：B 层边缘，交通、仓储一带；  
- 鬣狗帮：C 层豁口、废弃网络；  
- 花豹党：B 层核心居住区，掌握少量通往 C 层的路径情报；  
- 反抗军：多在 C 层，B 层可能有伪装节点。
