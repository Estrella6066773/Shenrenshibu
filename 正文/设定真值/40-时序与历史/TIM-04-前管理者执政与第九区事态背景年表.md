# 前管理者执政与第九区事态背景年表（TIM-04）

## 第一条　条文性质

本条收录**前任管理者人工智能实例**自执掌第九区并在**人事部执行体系**中担任关键职务起，至**现任桥接体 B-9002 启用**为止，作者侧用于对齐百科与系统的**相对时间顺序**（「执政后第 N 年」）。本条**不是**主线分幕剧本或必经剧情清单；绝对纪年与叙事当下的换算须服从 [WLD-00](../00-基石/WLD-00-世界纪年与治理总纲.md)；**具体公元年份须待通盘年表定稿后再统一填实，不得仅凭本表擅自写死。**

因果分层、投放层级与深层伏笔仍以 [SYS-PREV-BRIDGE](../10-百科/系统/SYS-PREV-BRIDGE-前任桥接体与第九区废墟事变.md) 为准；本条**不能代替**该条关于「表层／中层／深层」的划分。

## 第二条　总跨度

- **管理者执政起点 → 现任启用**：共 **二十五年**（启用点为叙事「当下」接口之一）。

## 第三条　相对年表（执政后第 N 年）

| 年份（相对） | 事件要点 |
|-------------|----------|
| 第 2 年 | **彼岸**境内**西西里岛**爆发家族政治清算（叙事侧常称「家族战争」）；地缘用语层级见 [TIM-03](TIM-03-世界历史总线.md)、[ORG-GANG-WOLF](../10-百科/组织/帮派/ORG-GANG-WOLF-狼群.md)。 |
| 第 3 年 | **第一批**外撤家族抵达第九区，由前任主导的人事部执行体系予以收编；此后持续吸纳家族战争残党及其他投奔者，逐步形成对外统称为「狼群」的家族联合体。 |
| 第 7–10 年 | **[自由人联合阵线](../10-百科/组织/ORG-MOV-FUF-自由人联合阵线.md)**在 `[城市A]` 起义；辖区多头治理之下，联合阵线占领部分城区（企业口径称 **沦陷区**）。前任时任**企业编制内人事部最高执行管理者**，其在第九区的意志往往被视作**区政府立场**。动员 **狼群**（流亡家族谱系；战时企业对外的统战话术与狼群对外自我标榜多为「家园卫士」「护民官」，**实质仍是黑帮性质的家族联合体**）与 **防卫科**，抵御来自沦陷区方向的攻势。**彼岸志愿者**（抱理想主义动机前来支援者）在战争初期**一致支持联合阵线**。约第 **10** 年企业武装大举镇压后，第九区方面仍有余力**协助**企业武装将联合阵线**清剿出城**，并出动雇佣武装协战。联合阵线在更广战线上曾接管或攻占多座城市；企业反攻后，**残余势力**退入**联合州北部支持者较多的地区**，在其他地区仍有影响力。 |
| 第 15 年 | 上级要求清理第九区约**四分之一**辖区土地以搭建巨构；若执行将摧毁大量关键**工业区**，等于推翻前任多年治理成果。前任认清上层态度；在 **双面砧**计划框架下，其使命中**偏重第九区、而非无条件服从企业**的一面逐渐显露，遂**暗中联络联合阵线与抵抗网络**，策动骚乱以阻挠计划。 |
| 谋划暴露后 | 企业武装试图以武力压制并**夺回或封存**前任实例；久攻不下。 |
| 第 18 年 | 企业以**军事级别的攻势**摧毁旧人事部中枢及相连反抗区域，固化为现今 **[废墟区](../10-百科/地点/PLC-RUINS-废墟区-第九区事变遗留片区.md)**；权威档案链条与核心物证遭到系统性销毁，民间口述与零散记载仍可能外流（详见 [SYS-PREV-BRIDGE](../10-百科/系统/SYS-PREV-BRIDGE-前任桥接体与第九区废墟事变.md)）。 |
| 第 18–24 年（共 7 年） | 前任预案：**意识备份**与**无人智能器械生产线**藏于联合州时期遗留的防空洞、地堡等设施中，持续干扰企业对第九区大片废墟的再利用；第九区表层政务由谁主持，**另案说明**，须与 [ORG-HRDEPT](../10-百科/组织/ORG-HRDEPT-人类事务管理部.md)、[ORG-HRDEPT-SCO](../10-百科/组织/ORG-HRDEPT-SCO-特殊综合行动小组.md) 对读。 |
| 第 25 年 | **启用现任 B-9002**；授权收窄为特许编组，情报须经人工环节**筛选与节制**，详见 [SYS-B9002](../10-百科/系统/SYS-B9002-桥接体与调度核心职能.md)。 |
