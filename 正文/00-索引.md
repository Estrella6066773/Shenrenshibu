# 世界观构建索引

本目录用于世界观长期构建与信息分级投放，不按剧情流程编排。

## 设计目标

- 以对象域知识建模作为正文骨架：确保知识体系具备可扩展性、可检索性和可校验性
- 基于受众视角的信息可见性作为投放机制：确保信息能够按照权限和阶段有序释放
- 将“事实内容”与“信息知情权”解耦，避免文档之间的相互污染
- 默认在正文段落中内嵌“可见性投放注记”，而非创建平行的文本体系

## 主目录

- `03-规则域/02-世界观总览与叙事锚点.md`：**世界观跨域总览**（`LORE-RULE-002`），内部圣经与导航；事实真值以各域词条为准，**勿直接全文投放给玩家**。
- `03-规则域/03-叙事隐喻与社会学参照.md`：作者向写作参照（`LORE-RULE-003`），非硬设定。
- `01-实体域`：人物、组织、地点、资源、技术、事件
- `02-关系域`：控制关系、依赖关系、冲突关系、交易关系、渗透关系
- `03-规则域`：法律法规、制度规范、灰色地带默契、惩罚机制
- `04-时序域`：时间节点、阶段状态、转移触发条件
- `05-观测域`：可见现象、指标数据、舆情动态、行为后果
- `90-可见性投放`：作者/角色/玩家/系统四类视角的信息等级划分
- `99-治理与模板`：词条模板、字段定义、维护流程规范

## 阅读路径（新人 / 协作 onboarding）

1. `03-规则域/02-世界观总览与叙事锚点.md`（`LORE-RULE-002`）— 地图与禁令
2. `03-规则域/01-终局约束与世界背景公理.md`（`LORE-RULE-001`）— 公理与术语
3. `04-时序域/03-世界历史总线.md`（`LORE-TIME-002`）— 全球史
4. `04-时序域/02-城市历史专区.md`（`LORE-TIME-003`）— 城市史与三群体
5. `01-实体域/组织/`（`LORE-ENT-ORG-003` 集团 → `LORE-ENT-ORG-001` 人事部 → `LORE-ENT-ORG-002` 狼群）与 `02-关系域`
6. `05-观测域/` — 可投放现象模板（`LORE-OBS-*`）

## 词条编辑顺序（建模 / 防双维护）

1. 先改**事实真值**：时序 `LORE-TIME-*`、实体 `LORE-ENT-*`、关系 `LORE-REL-*`
2. 再改**观测层** `LORE-OBS-*`（不造新设定，只归档可见后果）
3. 再同步**规则总览** `LORE-RULE-002` 的链接与禁令段（避免在总览内复述长文）
4. 最后检查各段 `[投放注记]` 与 `90-可见性投放` 出口约定

## 使用顺序（推荐）

1. 在对象域中录入事实内容（01-05）
2. 在正文关键段落中插入"投放注记"（受众/窗口/载体/偏差）
3. 使用治理模板进行一致性检查与评审（99）
